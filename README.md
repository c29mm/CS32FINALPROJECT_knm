# CS32FINALPROJECT_knm
CS32 Final Project w/ Spotify API 
*** For our final project, we initially wanted to predict what music a person wants to hear based on their previous listening history.
However, we have recently simplified it to a music recommender based on the genres a person likes and the keywords the like to hear.
For example, if someone likes to hear R&B love songs, what if they tried indie love songs instead? 
The computational task we would focus on is using the Spotify API (or something analagous) to look at song genres and lyrics, and then basically cross that info with what the user puts as their favorite genres and lyrics. The user would first input the lyrics or theme they would like to hear about. Then, the server would look at which songs have those lyrics and/or themes. Then, it would produce a list of songs and their genres. The user would then say whether they want something old or something new (from the genre they like or haven't tried yet) and get a song recommendation based on that choice. ***
